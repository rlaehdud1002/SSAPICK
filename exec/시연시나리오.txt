가이드 페이지
USER GUIDE 버튼을 눌러 가이드 확인

로그인
카카오 소셜 로그인 진행
Mattermost 인증 진행
힌트로 사용될 추가 정보 입력 진행

출석
어플 접속 후 자동으로 출석 체크 진행
출석 체크와 동시에 SSAPICK의 재화 서비스인 PICKCO 획득 가능
프로필 페이지에서 출석 현황 확인 가능

SSAPICK
NAVBAR의 가운데 있는 버튼을 눌러 SSAPICK 페이지 이동
화면 상단에는 질문 정보(카테고리, 질문 내용) 노출
화면 하단에는 선택할 수 있는 친구 목록 노출
하단에 나오는 4명의 친구 목록을 클릭하면 질문에 대한 SSAPICK 진행 가능
친구 목록 하단의 친구 목록 셔플 버튼을 클릭하면 PICKCO 10코인을 사용해 새로운 친구 선택지를 받을 수 있음
질문 오른쪽 하단의 질문 패스 버튼을 통해 질문 패스 가능
질문 왼쪽 하단의 질문 신고 버튼을 통해 질문 차단 및 패스 가능
총 10개의 질문에 응답하면 300개의 PICKCO 획득 가능

메인 페이지
NAVBAR 가장 왼쪽 버튼을 통해 메인 페이지 이동
메인 페이지에서 PICK 받은 모든 질문 확인 가능
아코디언을 열어 최대 2개의 힌트 확인 가능
공개 되지 않은 힌트는 ‘?’ 버튼을 클릭해 100개의 PICKCO를 사용하여 힌트를 얻을 수 있음
아코디언 오른쪽 하단의 쪽지 전송 버튼을 통해 20개의 PICKCO를 사용하여 쪽지 전송 가능
적절한 쪽지만 전송하기 위해 AI 필터링 후 쪽지 전송

위치 기반
PICKCO를 획득하기 위한 하나의 방법으로 위치 기반 서비스 제공
PICK 페이지로 이동하였더니 COOLTIME이 존재하므로 프로필 페이지의 내 주변 알림 페이지 이동
주위 사람을 클릭하면 10 PICKCO 획득 가능
하루 최대 10명까지 주위 사람 클릭 가능

쪽지함
NAVBAR의 오른쪽 2번째 버튼을 클릭해 쪽지함 이동
받은 쪽지함 – 내가 PICK한 사람들이 보낸 쪽지이기에 예측 가능하므로 실명으로 표기
보낸 쪽지함 – 누가 보낸건지 모르기 때문에 익명으로 표기
받은 쪽지 내용이 이상하면 쪽지 오른쪽의 점 3개를 클릭해 쪽지 차단 가능
쪽지를 차단하면 그 쪽지를 보낸 사용자가 차단되고, 그 사용자는 SSAPICK 선택지와 쪽지, 친구 목록 등에서 보이지 않게 됨

알람함
화면 오른쪽 상단의 알림 버튼을 클릭해 알림함 이동
받은 PICK, 쪽지 알람 및 질문 생성 알람 등 다양한 알람을 한 눈에 확인 가능
읽지 않은 알람과 읽은 알람 구분 가능
